Product Weight Barcode on Point Of Sale - POS Screen Odoo Apps

POS Screen Weight Barcode odoo app helps users to generate weight barcodes for products from point of sale screen.Users can configure product to generate barcodes and enter weight for selected product, product price will automatically calculated according to weight and generate weight barcode. Users can scan the weight barcode at the point of sale and product will be added with weight and price from the barcode.

Features

To Weight With Scale Option

Only configured product to weight with scale can only generate weight barcode.

Generate Weight Barcode from POS

User can generate weight barcode from point of sale screen

POS Weight Screen

On point of sale screen user can enter weight for product.

Scan Weight Barcode on POS

User can scan weight barcode and product with same weight and price will added to pos cart.
Demo Request Video Tutorial

Weight Scale Screen Configuration

User have to enable "Weight Scale Screen" option under point of sale session.

POS Product Weight Scale Configuration

User need to enable "To Weigh With Scale" option to generate weight barcode for product.

POS Weight Screen

On point of sale session user can see weight screen and select a product for that user wants to generate weight screen.

On selecting product user can enter weight for product and click "Set Weight" on debug window.

User can see entered weight will set and price will automatically calculated according to entered weight, User need to click "Generate" button on pos screen.

Product Weight Barcode

User can see generated product weight barcode.

Scan Weight Barcode

User can scan weight barcode from point of sale screen.

User can see on scanning weight barcode product with same weight and price will added to point of sale cart.
